---
title: "Quickstart"
description: "Go from zero to your first ranked GitHub repository search in under two minutes."
---

RepoHunt is a browser-based app — no installation required. All you need is a free Groq API key.

<Steps>
  <Step title="Get a free Groq API key">
    Go to [console.groq.com](https://console.groq.com) and sign up for a free account. Once logged in, create a new API key. Your key will start with `gsk_`.

    <Note>
      RepoHunt stores your key only in your browser's `localStorage`. It is never transmitted to any server other than Groq's own API when generating summaries.
    </Note>
  </Step>

  <Step title="Open RepoHunt">
    Visit the RepoHunt app at [github.com/Sujosh-Jana/RepoHunt](https://github.com/Sujosh-Jana/RepoHunt).

    On first load, a modal prompts you to enter your Groq API key. Paste your key and click **Save →** (or press Enter).

    <Tip>
      To update your key later, click the **⚙ change api key** button below the search bar at any time.
    </Tip>
  </Step>

  <Step title="Enter a search query">
    Type a query into the search bar and click **Search →** or press Enter. Be descriptive — RepoHunt passes your query directly to the GitHub search API and uses it to rank relevance.

    Not sure what to search? Click any of the quick-search chips below the search bar to run a pre-filled query:

    | Chip | Query sent |
    |------|-----------|
    | trading bot | `trading bot` |
    | admin dashboard | `admin dashboard HTML` |
    | Python backtesting | `Python backtesting` |
    | Unity save system | `Unity save system` |
    | AI agent | `open source AI agent` |
    | web scraper | `web scraper Python` |
    | realtime chat | `realtime chat app` |
    | markdown editor | `markdown editor` |
  </Step>

  <Step title="Apply filters (optional)">
    Click **Filters** to narrow results before or after searching. The following filters are available:

    | Filter | Options |
    |--------|---------|
    | **Language** | Python, JavaScript, TypeScript, Go, Rust, C#, Java, C++, Ruby, Swift |
    | **Min Stars** | Any, 50+, 100+, 500+, 1k+, 5k+ |
    | **Last Updated** | Any time, Last month, Last 3 months, Last 6 months, Last year |
    | **Exclude Forks** | Include forks, No forks |

    Filters are applied as GitHub search qualifiers, so results reflect the constraints before ranking.
  </Step>

  <Step title="Browse and sort your results">
    Results appear as ranked cards. Each card shows:

    - **Repo name and owner** — links directly to GitHub
    - **Language tag** — the primary language detected by GitHub
    - **Star count** and **fork count**
    - **Last updated** — how recently the repo was pushed to
    - **Repo Score badge** — a letter grade (S / A / B / C / D) with a numeric score from 0–100, calculated from stars, recency, description quality, topics, license, and query relevance
    - **AI summary** — a one-sentence analysis generated by `llama-3.1-8b-instant` via Groq, describing what the repo is useful for and its standout quality
    - **Topics** — up to five GitHub topic tags
    - **View on GitHub** link

    <Tip>
      The first result is labeled **top pick** — it is the highest-scoring repo for your query.
    </Tip>

    Use the **sort** controls in the results bar to reorder results:

    | Sort | Behavior |
    |------|----------|
    | **Score** | Ranked by RepoHunt's composite quality score (default) |
    | **Stars** | Sorted by total GitHub star count, descending |
    | **Recent** | Sorted by most recent push date, descending |
  </Step>
</Steps>
