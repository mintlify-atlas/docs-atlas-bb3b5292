## What is RepoHunt?

RepoHunt solves a real problem: GitHub search surfaces too many low-quality or irrelevant repositories. Instead of scrolling through hundreds of results, RepoHunt ranks repositories using multiple quality signals and generates AI summaries so you can quickly identify the best projects for your needs.

Search for something like `trading bot`, `Unity save system`, or `AI agent framework` — RepoHunt returns the top repositories ranked by quality, relevance, and activity, with an AI-generated one-liner for each.

## Why not just use GitHub search?

GitHub search is powerful, but it has limitations:

- It prioritizes **raw popularity** (stars) over quality
- It surfaces **irrelevant or abandoned repositories** alongside active ones
- It provides **no contextual summary** of what a project actually does or who it's for
- Niche discovery is **difficult without deep search knowledge**

RepoHunt combines ranking signals with AI to give you a curated, quality-first view of GitHub for any topic.

## Tech stack

RepoHunt is a single-file browser application — no backend, no build step, no dependencies to install.

| Layer | Technology |
|---|---|
| Frontend | HTML, CSS, JavaScript |
| Font | JetBrains Mono |
| GitHub data | GitHub Search API (unauthenticated) |
| AI summaries | Groq API (`llama-3.1-8b-instant`) |
| Key storage | Browser `localStorage` |

<Note>
Your Groq API key is stored only in your browser's `localStorage`. It is never transmitted to any server other than Groq's own API endpoint.
</Note>
